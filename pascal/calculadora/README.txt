Português:
Esse foi meu primeiro programa utilizando a linguagem Pascal. Estou me utilizando desta linguagem para aprimorar minha lógica de programação e estruturação de um código.

Como funciona o código:
O código basicamente está estruturado em quatro procedimentos (procedures) diferentes, uma para cada operação matemática básica. Além de um corpo principal que é o menu inicial mostrado ao usuário.
No menu, o usuário deve escolher entre cinco opções, soma, subtração, multiplicação, divisão ou sair.
Ao entrar em uma das opções matemáticas, o usuário deve informar com quantos números serão efetuados a operação e em seguida insere cada valor. O programa exibirá cada um dos números que estão sendo requisitados na operação e no final de tudo o resultado obtido. Após a operação o programa retorna para o menu principal.

English:
This was my first program using Pascal language. I'm using this language to improve my programming logic and code structuring.

How the code works:
The code basically is structured in four different procedures, one for each basic math operation, and a main body that works as a initial menu shown to the user.
In the menu, the user chooses between five options: addition, subtraction, multiplication, division or exit.
Entering in one of basic math operations, the user inputs how many numbers will be used then enters each value. The program displays each requested number inside the operation, and finally, the result. After the operation, the program returns to the main menu.
